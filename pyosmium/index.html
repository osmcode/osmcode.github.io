PyOsmium

Python bindings to Osmium Library

Features

This Python module allows you to access some of the features of the Osmium Library from Python code.

Code

The code is available from GitHub: github.com/osmcode/pyosmium.

Documentation

See the README.

The reference documentation is available for several versions:

  • latest (latest released version)
  • master (development version)

Download

Current and past releases can be downloaded from Github.

Packages are available for Debian Sid (also available in Jessie backports), Fedora and Archlinux.
