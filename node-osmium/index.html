Node-Osmium

Node.js bindings to Osmium Library

Features

This Node.js module allows you to access some of the features of the Osmium Library from Javascript code.

Note that currently this project is not maintained and its repository is archived.

Code

The code is available from GitHub: github.com/osmcode/node-osmium.

Documentation

See the README for build instructions etc. and look at the tutorial for usage instructions.
